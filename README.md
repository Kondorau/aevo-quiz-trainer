# 🎓 Universeller PDF Quiz-Trainer: Verwandelt strukturierte PDF-Fragenkataloge in interaktive Lern-Dashboards

Ein interaktives Dashboard zur Vorbereitung auf die AEVO-Prüfung (AdA-Schein). Dieses Tool wurde entwickelt, um statische Fragenkataloge aus PDFs in ein modernes, effizientes Lern-Erlebnis zu verwandeln.

## ✨ Features
- **PDF-Import:** Lade deinen Fragenkatalog direkt als PDF hoch.
- **Interaktives Dashboard:** Fixierter Fragenbereich und Live-Statistiken für optimale Übersicht.
- **Intelligentes Fehler-Log:** Falsch beantwortete Fragen werden automatisch in einer Liste gesammelt.
- **Fokus-Modus:** Trainiere gezielt nur die Fragen, die du zuvor falsch beantwortet hast.
- **Dark Mode:** Schonendes Lernen am Abend mit umschaltbarem Design.
- **Datenschutz:** 100% lokal. Keine Daten werden an Server übertragen; dein Lernstand wird nur in deinem Browser gespeichert.

## 🚀 Schnellanleitung
1. Öffne den Link zum Quiz-Trainer.
2. Klicke auf "Datei auswählen" und lade dein AEVO-Fragen-PDF hoch.
3. Beantworte die Fragen. Bei Fehlern erscheint rechts automatisch dein Lern-Log.
4. Nutze den Button **"💾 Export"**, um deine Fehlerliste als Textdatei zu sichern, oder **"🔥 Fehler üben"**, um Wissenslücken zu schließen.

## 🛠 Technik & Sicherheit
Dieses Tool ist eine "Client-Side-Only" Applikation.
- **Sprachen:** HTML5, CSS3 (Flexbox), Modern JavaScript (ES6+).
- **APIs:** [PDF.js](https://mozilla.github.io/pdf.js/) für die PDF-Verarbeitung im Browser.
- **Speicherung:** Nutzung des `LocalStorage` für die Speicherung der Erfolgsquote und Fehlerliste auf dem Endgerät des Nutzers.
